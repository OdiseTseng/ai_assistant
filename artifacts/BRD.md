# 業務需求文件 (BRD) - AI 通勤助手

**日期**: 2026-01-20

---

## 1. 執行摘要 (Executive Summary)
本專案旨在開發一款基於 Generative AI (Gemini) 的智慧通勤助手。解決現代上班族在「日常通勤」、「返鄉」及「臨時行程」中，面臨的大眾運輸轉乘複雜、天氣變數大及最後一哩路規劃困難等痛點。透過對話式 AI 與即時數據整合，提供「一鍵規劃」的極致體驗。

## 2. 商業價值與目標 (Business Value & Goals)
### 2.1 商業價值
*   **提升決策效率**：將原本需要查詢 Google Maps、公車 App、YouBike App 的多步驟流程，縮減為單一介面操作，節省 80% 查詢時間。
*   **個人化體驗**：透過記憶用戶偏好（住家/公司位置、交通工具習慣），提供比通用地圖更貼心的建議。
*   **技術展示**：作為 AI Agent 與前端技術整合的示範專案，展示 LLM 在實際生活場景的應用潛力。

### 2.2 市場目標
*   **目標受眾**：依賴大眾運輸的通勤族、跨縣市移動者、科技愛好者。
*   **關鍵成功指標 (KPI)**：
    *   API 查詢成功率 > 95%。
    *   使用者對路線建議的滿意度（透過減少重新查詢次數衡量）。
    *   頁面載入與互動延遲 < 2秒。

## 3. 專案範疇 (Project Scope)
### 3.1 核心功能 (In-Scope)
*   **三大情境儀表板**：日常通勤、回老家、想去哪 (Custom)。
*   **智慧路線規劃**：整合 Gemini AI 分析最佳路徑（考量天氣、時間）。
*   **最後一哩路支援**：整合 YouBike 2.0 即時站點資訊。
*   **多裝置支援**：響應式網頁設計 (RWD)，支援 Desktop 與 Mobile。
*   **個人化設定**：本地端儲存 (LocalStorage) API Key 與常用站點。

### 3.2 暫不執行 (Out-of-Scope)
*   即時導航 (Turn-by-turn navigation)。
*   票務訂購系統串接。
*   社群分享功能。
