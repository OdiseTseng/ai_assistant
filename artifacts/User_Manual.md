# 使用者手冊 (User Manual) - AI 通勤助手

**適用對象**: 終端使用者 (End Users)

## 1. 歡迎使用 (Introduction)
歡迎使用 **AI 通勤助手 (Antigravity Assistant)**！這是一個專為解決您通勤與出行煩惱而設計的智慧網頁應用。它能依據即時天氣、連假狀況以及您的個人偏好，為您規劃最舒適的上班、回家或出遊路線。

## 2. 快速開始 (Getting Started)

### 2.1 初次設定
1.  **開啟網頁**：首次進入時，會跳出「歡迎使用」視窗。
2.  **輸入金鑰**：請在欄位中填入您的 Google Gemini API Key（若無，請向管理員索取或自行申請）。
3.  **設定地點**：
    *   **上班地點**：點擊「設定」，輸入公司地址或關鍵字。
    *   **居住地點**：輸入住家地址。
    *   **最後一哩路**：設定當您抵達車站後，習慣搭乘什麼交通工具（公車/YouBike/走路）前往目的地。
4.  **儲存**：點擊「確認儲存」，系統將記住您的偏好。

### 2.2 日常功能
*   **上班/回家**：系統會根據時間自動判斷您要去上班還是回家。
    *   點擊畫面中間的 **「😩 艱難上班去」** 或 **「😃 快樂下班去」** 按鈕。
    *   稍等約 3-5 秒，AI 將為您列出建議路線。
*   **回老家**：
    *   點選上方 **「回老家」** 分頁。
    *   點擊 **「📍 回家囉」** 按鈕，系統將規劃跨縣市返鄉路線。
*   **想去哪？ (自訂行程)**：
    *   點選 **「想去哪？」** 分頁。
    *   在輸入框輸入目的地（例如：「淡水老街」、「台北 101」）。
    *   勾選您偏好的交通工具（捷運、火車、公車、YouBike）。
    *   點擊 **「🚀 出發」**，AI 為您規劃專屬行程。

## 3. 常見問題排解 (FAQ)

### Q1: 為什麼按鈕顯示「❌ 請先設定 API Key」且無法點擊？
**A:** 這代表您尚未設定 AI 金鑰。請點擊左上角的 **「設定 (⚙️)」** 圖示，在設定視窗中貼上有效的 API Key 並儲存。

### Q2: 為什麼顯示「AI 現正忙碌中 (503)」？
**A:** 這表示 Google 的 AI 服務目前使用量過大，這是暫時性的。請稍等 1-2 分鐘後再試一次即可。

### Q3: 為什麼找不到我要的地點？
**A:** 系統使用 OpenStreetMap 圖資。嘗試輸入更完整的名稱（例如「台北車站」取代「北車」），或加上縣市名稱（例如「桃園 中壢夜市」）會更精確。

### Q4: 如何更新我的最後一哩路設定？
**A:** 隨時點擊左上角 **「設定 (⚙️)」** 圖示，重新選擇您想要的交通方式接著儲存即可。

### Q5: 手機版如何打開選單？
**A:** 點擊左上角的 **「漢堡圖示 (☰)」** 即可展開側邊功能選單。

## 4. 隱私聲明 (Privacy Policy)
本系統 **不會** 將您的 GPS 精確座標或 API Key 上傳至任何第三方伺服器（除了必要的 Google Gemini API 請求外）。所有設定皆儲存在您瀏覽器的本地端 (LocalStorage)，請安心使用。
